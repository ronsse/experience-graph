# Agent Guide

Operational reference for AI agents interacting with the Experience Graph. These documents provide exact schemas, CLI commands, and step-by-step playbooks needed to read from and write to the shared experience store. They are framework-agnostic and applicable to any LLM-based agent.

## Contents

| Document | Description |
|----------|-------------|
| [trace-format.md](trace-format.md) | Complete reference for constructing and ingesting valid trace JSON |
| [schemas.md](schemas.md) | Machine-readable catalog of all Pydantic schemas with field definitions and examples |
| [operations.md](operations.md) | Full CLI command reference and Python-only mutation API |
| [playbooks.md](playbooks.md) | Step-by-step operational procedures for common agent tasks |
